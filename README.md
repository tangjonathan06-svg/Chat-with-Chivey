# Chat with Chive 🐶

A Streamlit chat app where you can talk to Chive the dog.

## Files

- `dog.py` — core logic: handles Chive's responses
- `dogfrontend.py` — Streamlit web UI
- `chive.jpeg` — Chive's profile picture

## Setup

Install dependencies:

```
pip install streamlit
```

## Running

```
streamlit run dogfrontend.py
```

## What to say to Chive

| You say | Chive says |
|---|---|
| walk walk | ARF ARF ARF ARF... |
| greenie | woof woof woof... |
| no more greenies | grrrrrr grrrrrr... |
| anything else | a random bark |
